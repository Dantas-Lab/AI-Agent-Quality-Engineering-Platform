# Protocolos de Atendimento

## Geração de Protocolo

Toda solicitação concluída com sucesso deve gerar um número de protocolo único.

## Consulta de Protocolo

O cidadão pode consultar o andamento de uma solicitação utilizando:

- número do protocolo;
- CPF do solicitante.

## Status Disponíveis

Uma solicitação pode apresentar os seguintes status:

- recebida;
- em análise;
- aguardando documentação;
- concluída;
- recusada.

## Segurança

O sistema não deve fornecer informações sobre um protocolo quando os dados de identificação apresentados não corresponderem ao solicitante.

## Atendimento Humano

Solicitações com inconsistências ou falhas de identificação devem ser encaminhadas para atendimento humano.
